# Dashboard Generator:https://dashboard-generator-u5vb.onrender.com

This project is a Python-based dashboard generator that allows users to analyze, clean, merge, and visualize CSV data files through a web interface. It is designed to streamline data processing and dashboard creation for business analytics.

## Features
- **CSV Analysis**: Analyze uploaded CSV files for structure and content.
- **Data Cleaning**: Clean and preprocess data to ensure consistency and accuracy.
- **Data Merging**: Merge multiple CSV files into a single dataset for comprehensive analysis.
- **Database Integration**: Store and retrieve processed data efficiently.
- **Web Dashboard**: Interactive web interface for uploading files, merging data, and viewing dashboards.
- **Static Assets**: Includes HTML, CSS, and JavaScript files for a modern UI.

## Project Structure
```
csv_analyzer.py         # Analyze CSV files
data_cleaner.py         # Clean and preprocess data
data_merger.py          # Merge CSV files
database.py             # Database operations
main.py                 # Main application entry point
render.yaml             # Configuration for rendering
requirements.txt        # Python dependencies
sample_orders.csv       # Sample data file
sample_sales.csv        # Sample data file
static/                 # Static web assets (HTML, CSS, JS)
merged/                 # Output merged CSV files
uploads/                # Uploaded files
```

## Getting Started
1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run the Application**
   ```bash
   python main.py
   ```
3. **Access the Dashboard**
   Open your browser and navigate to `http://localhost:5000` (or the configured port).

## Usage
- Upload CSV files via the web interface.
- Analyze and clean data using provided tools.
- Merge datasets for unified analysis.
- View results and dashboards in the browser.

## Folder Details
- **static/**: Contains HTML templates, CSS styles, and JavaScript files for the web UI.
- **merged/**: Stores merged CSV files generated by the application.
- **uploads/**: Temporary storage for user-uploaded files.
